Quais são as 20 linguagens de Programação mais populares do momento ?



Programadores precisavam ficar atentos as principais linguagens disponíveis e sempre é bom saber quais andam mais populares. Por isso, a empresa de análise RedMonk avalia um ranking de linguagens mais populares há alguns anos. Para criar o ranking, a firma faz uma correlação entre a base de dados do GitHub e do Stack Overflow. A ideia é encontra um equilíbrio entre o que é usado (GitHub) e o que é discutido (Stack Overflow).


A precisão da lista busca menos acertar a popularidade de uso e mais acertar os movimentos de uso.
O balanço do primeiro trimestre de 2020 saiu recentemente e pode apontar alguns novos caminhos e tendências:
  1. JavaScript

  2. Python

  3. Java

  4. PHP

  5. C#

  6. C++

  7. Ruby

  8. CSS

  9. Type Script

  10. C

  11. Swift

  12. Objective C

  13. Scala

  14. R

  15. Go

  16. Shell

  17. Power Shell

  18. Perl

  19. Kotlin

  20. Haskel

Algumas observações feitas pelo Redmonk sobre algumas das linguagens:

Python (+1): "Pela primeira vez na história desses rankings, o segundo lugar é de uma linguagem que não seja Java ou JavaScript. Mesmo que seja um empate. Por sua versatilidade, o Python contina a encontrar um papel para si mesmo em novos usos, além da acessibilidade e facilidade de uso."


TypeScript (+1): Com exceção do Swift por um único trimestre, faz cinco anos que não vemos alguma novidade entre os 10 melhores. Tendo realizado esse feito em junho, a questão era se o TypeScript poderia sustentar o desempenho. Subiu uma posição.


R (+2): O sucesso de R é um exemplo do poder de uma comunidade apoiada pela academia de elevar uma linguagem além do limite esperado.
